# 📘 Estructura del Módulo: 5 Niveles con Dificultad Progresiva

---

### 🔹 **Nivel 1: Principiante – Agrupar Objetos Visuales**

**Dificultad:** Baja  
**Objetivo:** Comprender intuitivamente qué significa agrupar elementos de diferentes maneras sin cambiar el total.

**Ejemplo:**
- Se muestran tres animales: 🐶, 🐱, 🐷
- Primero se agrupan así: (🐶 + 🐱) + 🐷
- Luego se agrupan así: 🐶 + (🐱 + 🐷)
- Pregunta: "¿En ambos casos hay el mismo número de animales?"

**Actividad interactiva:**
- El niño arrastra los animales para formar distintas agrupaciones.
- Se muestra que el total no cambia.

**Enfoque pedagógico:**
- Desarrolla la noción de agrupamiento flexible de forma visual e intuitiva.
- Base para entender la propiedad asociativa sin números aún.

---

### 🔹 **Nivel 2: Elemental – Introducción Numérica con Sumas**

**Dificultad:** Baja-Media  
**Objetivo:** Relacionar el agrupamiento visual con operaciones numéricas usando paréntesis.

**Ejemplo:**
- Se presenta:
  - `(2 + 3) + 4 = ?`
  - `2 + (3 + 4) = ?`

- Actividad: Resolver ambas expresiones y ver que dan el mismo resultado.

**Visualización:**
- Uso de colores o marcos para mostrar cada grupo dentro del paréntesis.

**Enfoque pedagógico:**
- Introduce el uso de paréntesis como herramienta para agrupar.
- Enseña que el orden de agrupar no afecta el total.

---

### 🔹 **Nivel 3: Intermedio – Aplicar la Propiedad en Ejercicios Guiados**

**Dificultad:** Media  
**Objetivo:** Practicar la propiedad asociativa con ejercicios estructurados y guiados.

**Ejemplo:**
- Se muestra:
  - `(6 + 1) + 3 = ___`
  - Completa la otra forma: `6 + (___ + ___) = ___`

**Otra actividad:**
- Elegir la opción correcta:
  - ¿Cuál es igual a `(4 + 5) + 6`?
    - a) `4 + (5 + 6)`
    - b) `(4 + 6) + 5`
    - c) `4 × (5 + 6)`

**Enfoque pedagógico:**
- Refuerza la identificación de expresiones equivalentes.
- Promueve el razonamiento deductivo simple.

---

### 🔹 **Nivel 4: Avanzado – Resolver Problemas Verbales y Mentales**

**Dificultad:** Media-Alta  
**Objetivo:** Usar la propiedad asociativa para facilitar cálculos mentales y resolver problemas.

**Ejemplo 1 (cálculo mental):**
- Resolver: `8 + 2 + 5`
- Guía: "Agrupa los números que suman 10 primero"
- Posible solución: `(8 + 2) + 5 = 10 + 5 = 15`

**Ejemplo 2 (problema verbal):**
- "Luis tiene 7 canicas, Ana tiene 3 y Carlos tiene 6. ¿Cómo puedes agruparlos para hacer la cuenta más fácil?"

**Enfoque pedagógico:**
- Enseña a usar la propiedad como estrategia útil.
- Fomenta el pensamiento estratégico y rápido.

---

### 🔹 **Nivel 5: Experto – Crear y Justificar Expresiones Equivalentes**

**Dificultad:** Alta  
**Objetivo:** Demostrar dominio completo aplicando la propiedad de forma autónoma y creativa.

**Ejemplo 1:**
- Se da un total objetivo: **30**
- Y tres números: **9, 6, 15**
- El niño debe crear dos expresiones asociativas que den 30:
  - `(9 + 6) + 15 = 30`
  - `9 + (6 + 15) = 30`

**Ejemplo 2:**
- Pregunta abierta: "¿Es cierto que `(a + b) + c` siempre da lo mismo que `a + (b + c)`? ¿Por qué?"
- Espacio para responder con dibujos, palabras o números.

**Enfoque pedagógico:**
- Desarrolla habilidades de razonamiento abstracto y justificación matemática.
- Es ideal para evaluar comprensión completa del concepto.
